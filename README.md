# The Laws Bridge

Le pont des lois est un projet visant à lister les lois en cours de
traitement et récement traitées.

Le code est à stade très approximatif mais la liste des lois en cours
(pour la 13ieme session) fonctionne.

A partir de l'url de publication un fichier haml puis html est généré.
Le but est d'insérer les données dans une base de données et de
permettre un parcours simple de chaque texte et de ses annexes.
